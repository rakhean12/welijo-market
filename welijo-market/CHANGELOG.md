# Changelog - WELIJO Market

Semua perubahan penting pada project ini akan didokumentasikan di file ini.

## [1.0.0] - 2026-03-12

### ✨ Fitur Awal
- Website marketplace sayur organik dengan desain modern
- Responsive design (mobile, tablet, desktop)
- Integrasi dengan Google Spreadsheet sebagai database
- Katalog produk dinamis dengan animasi
- Integrasi WhatsApp untuk pemesanan
- Sticky navbar dengan smooth scrolling
- Mobile menu toggle
- Loading state dan error handling
- Data produk dummy untuk development

### 🎨 Design
- Tema warna hijau, putih, dan kuning
- Font Poppins untuk tampilan modern
- Card produk dengan efek hover dan zoom
- Animasi fade-in untuk produk
- Shadow dan rounded corners
- Gradient background pada hero section

### 📱 Sections
- Navbar dengan logo dan menu navigasi
- Hero section dengan CTA button
- Katalog produk (grid responsif)
- Section tentang WELIJO Market
- Section kontak (WhatsApp, alamat, email)
- Footer dengan social media links

### 🛠️ Teknologi
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Google Apps Script API
- Google Spreadsheet

### 📚 Dokumentasi
- README.md - Dokumentasi utama
- PANDUAN-SISWA.md - Panduan lengkap untuk siswa SMK
- QUICK-START.md - Panduan cepat memulai
- google-apps-script.js - Kode backend API
- contoh-data-produk.csv - Template data produk

### 🎓 Fitur Pembelajaran
- Kode dengan komentar lengkap
- Struktur folder yang rapi
- Pemisahan HTML, CSS, dan JavaScript
- Contoh integrasi API
- Best practices untuk web development
